Prompting for Bioactivity

Molecular language models trained on SMILES strings can generate chemically valid, structurally novel molecules at scale. The open question is how to direct that generation toward molecules with desired properties without retraining the model for each target. The standard approach is iterative: generate candidates, score them, fine-tune the model on high-scoring examples, repeat. This works, but it is expensive and it produces a separate model for every target, which does not scale well when the target space is large or resources are limited.

The approach we explore here is different. Rather than updating the model, we ask whether the prompt alone can do the steering. The hypothesis is that a pre-trained generative transformer, prompted with molecules whose structural similarity to other compounds is strongly predictive of those compounds' bioactivity, will generate candidates that are, on average, more bioactive than molecules prompted randomly or from the full assay. If this holds, the same base model can be redirected to any target by changing only the input, with no retraining required.

The theoretical motivation comes from the Similarity Property Principle, a well-established empirical regularity in medicinal chemistry: structurally similar molecules tend to share biological activity. The question is whether the principle can be operationalized computationally to identify, within a given assay, which specific molecules are the best structural anchors for generative exploration.

Standard approaches select the highest-measured-activity compounds as prompts. This conflates two different things. A molecule with high measured bioactivity is not necessarily a molecule whose neighborhood in chemical space is enriched for bioactivity. What we actually want are molecules whose structural context is informative about bioactivity across the assay: molecules such that knowing similarity to them predicts the activity of other compounds.

The distinction matters because a generative model prompted with a molecule will explore the neighborhood around that molecule. If the neighborhood is bioactivity-enriched, the generated candidates are more likely to be active. If it is sparse or unstructured with respect to activity, the prompt provides weak guidance regardless of the seed molecule's own activity.

The pipeline identifies these structurally informative molecules using a Random Forest regressor trained on pairwise molecular similarity. Given a ChEMBL assay of n molecules, we compute an n×n Tanimoto similarity matrix using Morgan fingerprints, where entry (i, j) represents the fingerprint similarity between molecules i and j.

Fig. 1 — Identifying structurally informative prompt molecules from a ChEMBL assay. Each molecule's row in the similarity matrix serves as its feature vector; Random Forest feature importance identifies which molecules' similarity profiles best predict bioactivity.

A Random Forest regressor trained on this representation learns which molecules' similarity profiles are predictive of bioactivity. Feature importance analysis produces Gini importance scores for each molecule, reflecting how much variance in bioactivity the model attributes to similarity to that molecule. The top-scoring molecules are the ones whose structural neighborhood is most informative about activity in the assay. These are the prompt molecules.

This framing has a specific advantage over selecting by raw bioactivity. It is explicitly relational: it asks not "how active is this molecule?" but "how predictive is this molecule's structural context for activity across the assay?" A molecule can rank highly while having moderate measured activity, if its neighbors cluster strongly by activity level.

These molecules serve as prompts to a pre-trained GPT to generate novel SMILES strings. Generated structures are filtered for chemical validity using RDKit, and the remaining candidates are scored for predicted bioactivity using the fitted Random Forest model.

Fig. 2 — Scoring generated candidates. Each new SMILES is compared against all assay molecules to produce an n×1 similarity vector, which the fitted Random Forest uses to predict bioactivity.

The case study target is CBL-B, an E3 ubiquitin ligase involved in the downregulation of T cell receptor signaling. We obtained two independent ChEMBL assays for CBL-B, fit separate Random Forest models to each, and used the top-importance molecules from Model 1 as prompts to generate candidates. The evaluation design uses cross-assay prediction as a check against overfitting.

Fig. 3 — Cross-assay evaluation design for CBL-B. Candidates generated using Model 1's feature importance are scored by both Model 1 and Model 2 independently. Agreement between models trained on different assays provides evidence against overfitting.

The generated candidates score higher than the Assay 1 baseline under Model 1, with a p-value of 1.46 × 10⁻³⁵. They also score higher than the Assay 2 baseline under Model 2, and higher than the Model 2 predictions for the original Assay 1 molecules, both with p-values below 10⁻³⁰⁰. The cross-assay result is the more meaningful of these: candidates selected using Model 1 score higher than the original assay molecules under Model 2, which was trained on an entirely different set of measurements.

Several limitations are worth stating directly. The bioactivity predictions are model-based, not experimentally measured. The validation chain is: Random Forest trained on assay data, used to score generated molecules, compared against the same assay data. A candidate that scores well under this pipeline may still fail in a binding assay or a cell-based assay, for reasons that neither fingerprint similarity features nor the Random Forest model captures. Pharmacokinetic properties, synthetic accessibility, and off-target effects are not considered.

The cross-assay validation partially addresses the circularity concern, but both assays measure the same target, so they are not fully independent. A more stringent test would involve synthesizing candidates and measuring activity experimentally. The p-values reported reflect statistical comparisons within the space of model predictions, not empirical claims about in vitro or in vivo activity.

The circular risk in feature importance is also worth noting. The Gini importance score for molecule i reflects how much the Random Forest uses similarity to i when predicting bioactivity in the training assay. High-importance molecules are those whose structural neighborhood is well-represented and whose similarity profile correlates with the target variable. If the assay has sampling biases, those biases will be reflected in which molecules rank highly.

The core contribution is the use of Random Forest feature importance on a pairwise similarity matrix as a method for identifying structurally informative prompt molecules, as distinct from simply selecting high-activity compounds. The selected molecules are anchors in activity-predictive regions of chemical space, not just the highest-activity compounds observed in a particular assay. Whether this translates to experimental activity in synthesized candidates is the question that motivates the next stage of the work.
